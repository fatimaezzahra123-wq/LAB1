# HelloToast 
Application Android simple développée dans le cadre du cours **Programmation Mobile – Android avec Java**.

## Description
HelloToast est une application de démonstration qui illustre les concepts de base du développement Android :
- Affichage d'un message **Toast**
- Gestion d'un **compteur** interactif

## Interface
- Fond jaune vif pour la zone du compteur
- Boutons bleus en haut et en bas de l'écran
- Compteur affiché en grand au centre

## Technologies utilisées
- **Langage** : Java
- **IDE** : Android Studio
- **API minimum** : 24 (Android 7.0)
- **Interface** : XML (LinearLayout)

## Concepts appris
- Création d'un projet Android avec **Empty Activity**
- Conception de l'interface avec **XML**
- Liaison XML ↔ Java avec `findViewById()`
- Utilisation des **listeners** (`setOnClickListener`)
- Affichage d'un **Toast**
- Mise à jour dynamique d'un **TextView**
